Lesson 3: Comments and Attributes

What are comments?

Comments are text hidden within an HTML document. In order to create an HTML comment, you must use the characters “<!-->” before the comment text and “-->” after it Comments are not visible on the actual page, only in the code. They are used to tell other programers things about the code.

<!-- this is a single line comment: --> <!-- this is a Multiple line comment --> <!-- You can disable elements <h1>Hello World</h1> by enclosing them in a comme

Attributes are codes that appear within a tag that modify it in some way. The general syntax for an attribute is: <tagName attributeName=”value”> content</tagName>

What are links?

Links use the anchor or <a> element to make a hyperlink to another page or to another location on the same page. It is an inline element. To locate the link, you need the href attribute

  • The general syntax for a link is <a href= “partToFile”>LinkText</a>
  • >Absolute or external links: <a href=”https://google.com”>Google</a>
  • Relative or internal link: <a href= “/index.html”>HomePage</a>
Download: lets the user download a file <a download=”fileName”>... Target: specifies a specific piece within the page, the link opens Some targets:
  • blank opens in a new window/tab
  • top opens at the top of the page
  • self opens within the same iframe
  • parent opens within the parent of the current frame <a href=”URL” target=”_blank”>...
Example <a href=”https://www.youtube.com/watch?v=P2CzCFpz83Y”>Oompa Loompa Music Video</a> Oompa Loompa Music Video

Embedding Images:

Images are embedded on a web page using the <img< element. It is an empty element (no trailing tag) and works inline. To locate the image you must use the src attribute. You must also use the alt attribute to describe the image for screen readers or if the image will not load.

  • Local image: <img src=”happyface.jpg”alt=”Smiley Face”>
  • Remote image: <img src= “https://ex.com/hf.jpg”alt= “Smiley Face”>
How to Size Images: You can control the size of images using the width or height attributes, or both. The actuarial dimensions are specified in pixels.If you only specify the height or width, the image will scale proportionally. If you specify both, the image will fix itself to the dimensions you chose.

<img src=”file”width=”200”> (The style attribute can also be used) Example <img src=”Ommpa_Loompa.png” alt=”Oompa Loompa”>

Headings:

  • Headings are used to create an outline: structure of a web page
  • There are 6 headings types: <h1> through <h6>
  • Aside from the size of the text headings allow assistive devices like screen readers
    • All headings are block level elements <h1> The top-level heading</h1>
    <h2>second level</h2>
    <h3>third level</h3>
    <h4>fourth level</h4>

Paragraphs:

Paragraph <p> is the workhouse tag of web design. Most blocks of text use a paragraph for their formatting.Paragraphs are block-level elements, creating space above and below them on the page.If you want to create a line break, use the <br> tag. <P> most text in the web pages are formatted in paragraphs <br> is an inline tag.

Horizontal Rule and Preformatted Text

  • If you want to create a line between blocks of text, use the <hr> element. <hr> is an empty tag
  • If you want to preserve multiple spaces or line breaks for text like in poems, you can use the preformatted text <pre> tag. <pre> is a block-level element
