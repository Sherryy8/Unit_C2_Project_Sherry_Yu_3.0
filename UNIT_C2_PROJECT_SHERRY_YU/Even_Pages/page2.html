Lesson 1: HTML

What is HTML?

HTML stands for Hypertext Markup Language, the language that makes the World Wide Web possible. HTML was invented by Tim Berners Lee in 1989. A markup language formats text, such that when displayed, the markeups do not appear in HTML, the markups are called tags and are shown with the < or >

Basic HTML code:

A properly written web page should have the following elements:

  • The HTML’s doctype declaration
  • The html tag begins and ends the page
  • The head tag holds the page’s metadata
  • The title tag gives the words that show up on your browser tab
  • The body tag contains the visible content of the page

    • Where can you write HTML?

    • On Windows, you can edit HTML using NotePOadProgram.
    • On Mac, you can edit HTML using TextEdit program.
    • You can also use browser sites to edit HTML. (This is what a Chromebook would use.)

    EXAMPLE:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Title of the web page </Title >
    </head>
    <body>
    </body>
    </html>

    Lesson 2: Formatting Tags

    Types of Tags:

    HTML has two types of body element:

    • Block-level elements will automatically appear on separate files on the page
      • ex. <p> and <h1>
    • Inline elements can appear next to one another horizontally so that test with an inline element in it will not start a new line
      • ex. <strong> (turns text bold) and <i> (turns text into italics)

    Formatting Tags

    Several elements modify the look of text within them. These are all the inline elements, you can use them to modify one or more characters within a line of text.

    • <b> this will appear bold </b>
    • <strong> this will appear bold </strong>
    • <i> this will appear in italics </i>
    • <em> this will appear in italics </em>
    • <mark> this will appear highlighted </mark>
    • <small> this will appear smaller </small>
    • <del> this will appear crossed out </del>
    • <ins> this will appear underlined </ins>
    • <sub> this will appear as a subscript </sub>
    • <sup> this will appear as a superscript </sup>

    List Elements:

    There are two types of list elements:

    • <ol> creates an ordered list (a list with numbers)
    • <ul> creates an unordered list ( a list with bullets instead of numbers)
    Both list types use <li> (list item) for items within a list ol, ul, and li are block level domains

    EXAMPLE:

    <!DOCTYPE html>
    <html>
    <head>
    <title>Unit C2 Lesson 2 -Sherry Yu </title>
    </head>
    <body>
    <h1>Unit C2 Lesson 2 - Sherry Yu</h1>
    <h2>My School Day</h2>
    <h3><ins>My Class Schedule</ins></h3>
    <ol>
    <li><em>Global History</em></li>
    <li>English II </li>
    <li><mark>Health/Pre-ib Computer Science</mark></li>
    <li><mark> Phys-Ed/Pre-ib Computer Science</mark></li>
    <li><ins>Physics</ins></li>
    <li><strong>LUCNCH</strong></li>
    <li>Spanish II</li>
    <li>Algebra II</li>
    </ol>
    </body>
    </html>
